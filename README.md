# E-commerce-database

🚀 **Built an End-to-End E-Commerce Database in MySQL!** 🛒💻

As part of my SQL learning journey, I designed and implemented a complete **E-Commerce Database Management System** using MySQL.

### 📌 Database Modules Included:

✅ Customers
✅ Categories & Sub-Categories
✅ Products Inventory
✅ Orders Management
✅ Order Items Tracking
✅ Payments Processing
✅ Shipping Details

### 🔥 Key SQL Concepts Used:

🔹 Database & Table Creation (DDL)
🔹 Primary Keys & Foreign Keys
🔹 Auto Increment IDs
🔹 Constraints (NOT NULL, UNIQUE, CHECK)
🔹 One-to-Many Relationships
🔹 ENUM Data Types
🔹 Sample Data Insertion (DML)
🔹 Relational Database Design

### 📊 Real-World Features:

✔ Customer Registration
✔ Product Catalog Management
✔ Order Tracking System
✔ Payment Status Monitoring
✔ Shipping Information Management
✔ Inventory Control

This project helped me strengthen my understanding of **Database Design, Normalization, Relationships, and SQL Best Practices**.

Every database tells a story, and this one simulates the backbone of a modern online shopping platform. 🚀
